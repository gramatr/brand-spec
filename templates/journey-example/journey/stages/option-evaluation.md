# Stage — Option evaluation

The HR leader is no longer asking whether the problem is real; they
are asking which approach to take. The hurdles cluster around vendor
sameness and exec-team translation, not around problem belief. Content
that wins here helps the prospect *explain their decision upward*, not
content that explains the product downward.

## Authorship notes

The `mental_vocabulary` here is a place to be especially honest about
prospects using vendor-marketing language back to the brand. Phrases
like "single source of truth", "predictive", and "AI-powered" enter
the prospect vocabulary because every vendor uses them — which is why
they no longer differentiate. Capture the prospect language as it
actually shows up; let the content team decide separately whether
echoing those phrases is on-brand.
