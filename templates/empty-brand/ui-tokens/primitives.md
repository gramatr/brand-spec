# UI Tokens — Primitives (SAMPLE)

> **This is a sample file.** Replace with your brand's real primitive
> token palette. Primitives are the raw values — colors, spacing
> steps, font sizes — with no semantic meaning. The semantic layer
> (`semantic.md`) gives them meaning. Components consume semantic, not
> primitive. Delete this callout when you replace the content.
